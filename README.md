# 🛒 GoKart - E-Commerce Website

## 📌 Description
GoKart is a responsive e-commerce web application built using HTML, CSS, and JavaScript. It displays products dynamically using an external API.

## 🚀 Features
- Product listing using API
- Search functionality
- Responsive design with navbar
- Product details page
- Cart UI

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 📂 Project Structure
- Login Page
- Registration Page
- Home Page
- About Page
- Contact Page
- Product Details Page
- Cart Page

## 🌐 API Used
https://dummyjson.com/products
